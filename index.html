The HUNER tagger

We provide the HUNER tagger as an easy to use, integrated Docker container. HUNER is able to tag the five entity types Cell lines, Chemicals, Diseases, Genes/Proteins and Species. The tagger and usage instructions can be obtained from the HUNER Git repository. Model files are available at this folder.

The following table gives an overview of the performance (as F1-score) of HUNER on the CRAFT corpus and a comparison to the state of the art tools GNormPlus und tmChem.
Entity type SotA HUNER
Chemicals 38.25% 42.95%
Genes/Proteins 49.95% 59.37%
Species 77.29% 84.40%
